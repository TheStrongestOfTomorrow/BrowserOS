### Terminal
A powerful command-line interface. Beyond standard shell commands, it now features:
- **Python Integration**: Execute Python scripts directly using the `python` command, powered by Pyodide.
- **System Info**: Use `neofetch` to see a summary of your virtual hardware.

### File Explorer
Your window into the browserOS file system. Navigate through directories, view file properties, and manage your virtual data with a clean, intuitive interface.

### System Monitor
Real-time tracking of your virtual resources. Monitor CPU and RAM usage through dynamic, animated charts.
